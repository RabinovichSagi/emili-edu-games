# משחק בעיות מילוליות — תיעוד מוצר וטכני

## מטרת הלמידה

המשחק מחזק פתרון בעיות מילוליות בחשבון לכיתה ג׳: הילד קורא סיפור קצר, מזהה את הקשרים בין הכמויות, בוחר בראש את הפעולה המתאימה ומקליד תשובה סופית אחת בעזרת לוח מספרים גדול.

## החלטות יישום

- **אורך סשן:** 4–10 שאלות, ברירת מחדל 6.
- **תשובה:** תיבת תשובה אחת בלבד. אין בחירה מרובה.
- **קלט:** לוח מספרים מובנה, כדי להתאים לטאבלט ולמנוע תלות במקלדת.
- **חזרות מרווחות:** פריט חזרה מוגדר לפי `patternId + templateId`, כדי לתרגל גם מבנה מתמטי וגם ניסוח סיפור.
- **קושי:** `קל`, `רגיל`, `אתגר`; הקושי משנה טווחי מספרים, תוך שמירה על מספרים מתאימים לכיתה ג׳.
- **משוב:** טעות ראשונה מציגה רמז ומעודדת קריאה חוזרת; אחרי טעות נוספת מוצגת התשובה והפריט מסומן לחזרה קרובה.
- **אנטי־ניחוש:** הקלדה מהירה מדי בלוח המספרים מפעילה רמז עדין: לעצור ולחשוב לפני שמקלידים.
- **תשובות כן/לא:** בתבנית “האם יש מספיק?” מקישים `1` עבור כן ו־`0` עבור לא.

## מבנה הקבצים

- `game.js` — ממשק המשחק, סשנים, לוח מספרים, משוב, דיווחים וחיבור לחזרות מרווחות.
- `patterns.js` — מטא־דאטה של 20 דפוסי הבעיות.
- `numberGeneration.js` — יצירת מספרים, אכיפת מגבלות וולידציה לכל דפוס.
- `templates/` — ספריית תבניות, קובץ נפרד לכל דפוס בעיה; כרגע יש 8 תבניות לכל דפוס, כולל תבניות מעולמות ספיידי וחברים ובית הבובות של גבי.
- `PATTERNS.md` — מדריך דפוסים מפורט למורים ולסוכני קוד.
- `SCENARIO_WORLDS.md` — עולמות וסביבות בעברית לשימוש עתידי ביצירת תבניות נוספות.

## הוספת תבנית חדשה

1. בוחרים דפוס מתוך `patterns.js`.
2. בוחרים עולם וסביבה מתוך `SCENARIO_WORLDS.md`, כדי לשמור על אוצר מילים עקבי וטון מתאים.
3. פותחים את הקובץ המתאים תחת `templates/<patternId>.js`.
4. מוסיפים אובייקט עם `id`, `patternId`, ו־`text`.
5. משתמשים רק במשתנים שהמחולל של אותו דפוס מספק, למשל `{a}`, `{b}`, `{groups}`, `{size}`.
6. מוודאים שהניסוח בעברית טבעית, שהשאלה מבקשת תשובה מספרית אחת, ושאין רמז מילולי שחוזר תמיד באותו ניסוח.

## משתני תבניות נפוצים

- חיבור: `{a}`, `{b}`.
- חיסור: `{whole}`, `{part}`.
- חלק חסר: `{whole}`, `{known}`.
- השוואה: `{small}`, `{large}`.
- קבוצות שוות: `{groups}`, `{size}`.
- מערכים: `{rows}`, `{columns}`.
- חילוק שווה: `{total}`, `{groups}`.
- כמה קבוצות: `{total}`, `{size}`.
- השוואה כפולית: `{base}`, `{multiplier}`.
- דו־שלבי: משתנים לפי הדפוס, למשל `{start}`, `{increase}`, `{decrease}`, `{extra}`, `{removed}`.
