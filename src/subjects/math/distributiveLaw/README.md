# חוק הפילוג — החלטות מוצר

- **מטרת למידה:** פירוק מספר דו־ספרתי לעשרות ואחדות ושימוש בחוק הפילוג כדי לפתור כפל.
- **מבנה סשן:** 6 תרגילים כברירת מחדל, עם אפשרות ל־4/6/8/10 תרגילים.
- **פריט חזרה:** `a|b|tens+ones`, כלומר תרגיל ספציפי עם פירוק עשרות/אחדות.
- **קושי:** טווח המספר הראשון, טווח ספרת העשרות, וטווח ספרת האחדות; יש קיצורי דרך מתחיל/מתקדם.
- **פידבק:** פיצול לעשרות ואחדות מקבל חיזוק; פיצול אחר מונפש בעדינות ומציע רמז ללא ענישה.
- **אנטי־ניחוש:** ניסיונות שגויים ומהירים מציגים הודעת האטה ורמז עשרות.
- **אינטראקציה:** בחירת קו חיתוך על הגריד, הפרדה/חיבור מגנטיים, ואז מילוי משוואה באמצעות קלפי מספרים.
- **מדדים:** דיוק, זמן תגובה, חזרות מרווחות, ומאסטרי שמחושב מהיסטוריית הפריטים.
