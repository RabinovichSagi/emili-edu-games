# English — Case Typing (כתיבת מילים גדולות/קטנות)

- This game: `?subject=english&game=case_typing`
- Primary implementation: `src/subjects/english/caseTyping/game.js`

Learning goal:
- Practice converting between lowercase/uppercase letters while typing whole words.

Gameplay:
- A word is shown in one case (all lower / all upper).
- The keyboard shows letters in the opposite case.
- The child types the converted word and presses "שלח ✅".
